# DigitalShopBot — магазин цифровых товаров в Telegram

Telegram-бот для продажи цифровых товаров: ключи активации, PDF-гайды, подписки, промокоды.

## Возможности

- Каталог с категориями товаров
- Карточка товара с описанием и ценой
- Корзина и оформление заказа
- Автовыдача товара после оплаты
- База покупок

## Стек

- Python 3.10+
- Aiogram 3.x
- SQLite
- Инлайн-клавиатуры

## Установка

```bash
git clone https://github.com/pipupip/digital-shop-bot
cd digital-shop-bot
pip install -r requirements.txt
# настройте config.py
python bot.py
```
